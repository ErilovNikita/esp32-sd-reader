# V1 enclosure models (Older)
![Enclosure preview](default.png)

Older three-part enclosure version with a separate ESP32 holder.

## Files
| File | Description |
| --- | --- |
| `Case.stl` | Main enclosure body |
| `ESP Holder.stl` | Internal ESP32 holder |
| `Top.stl` | Top cover |

## Assembly
1. Print all three parts.
2. Place the ESP32 into the holder.
3. Install the holder into the case.
4. Close the enclosure with the top cover.
